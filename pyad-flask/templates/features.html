  • - Man kann einen Benutzer hinzufügen. Dieser bekommt automatisch die richtige Gruppe nach Auswahl der Abteilung und die Remotedesktopbenutze Gruppe zugewiesen. Ebenfalls wird dem Benutzer die richtige OU und einem Home-Laufwerk zugewiesen.
  • - Man kann einen Benutzer löschen. Dieser wird aus allen Gruppen in der er sich befindet entfernt.
  • - Sollte ein Benutzer die Abteilung wechseln, kann man dies hier auch realisieren. Dem Benutzer wird die Alte Gruppe, nach Auswahl der richtigen Abteilung aus der er raus wechselt, entfernt, wird in die neue richtige OU verschoben und ihm wird die neue Gruppe, nach Auswahl der richtigen Abteilung, zugewiesen.
  • - Wenn ein Kunde oder ein Mitarbeiter eine Berechtigung für einen speziellen Ordner braucht oder entfernt bekommen soll, kann das hier eingestellt werden. Dem Benutzer wird die entsprechende Gruppe, nach Auswahl auf der Seite, zugewiesen oder entfernt. Das kann nützlich sein wenn ein Kunde Zugriff auf den Projekte-Ordner benötigt.
